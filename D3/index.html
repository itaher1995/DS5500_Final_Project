Introduction

Welcome to "How Does VGG Learn?", an interactive visualization designed to whitebox a blackbox using effective and expressive visualizations. The user will able to explore several visualizations that will hightlight characteristics that VGG learns up until various stopping points. You will read some quick instructions on how to use this platform. At the bottom of this page you can click "Begin Visualization" to explore VGG's output from intermediate max pooling layers.

Toggle Functionality

The user has the ability to toggle between different max pooling layers to see the output up until that layer.

Exploring Intermediate Layer Outputs Using Clustering

The first primary visualization is the scatter plot, which plots 2993 images in two dimensions after various max pooling layers.

These images are clustered using K-Means. The user can zoom and pan the visualization, which will expose different parts of the visualization.

The user can hover over each point to get the cluster label and the ground truth class label. They can also click on a point and get the closest classes to that point.

















Explore Class Distribution, Filter Points Corresponding to a Class and Show Clustering Distribution

The second primary visualization is a bar plot. This bar plot plots the distribution of all 256 classes in the dataset that is currently visualized.

The user can scroll using the minibar on the bottom to see the frequency of each class.

Hovering on each main bar allows the user to see where the points corresponding to that class are on the scatterplot.

The user can also click on the actual bars to see the distribution of images across each cluster.
